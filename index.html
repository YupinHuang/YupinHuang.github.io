A9 search组面经

上学期我都没听过这个公司，它也从未出现过我的dream company list里。但是就这样，我最后神奇的来到了这家公司。上学期在LTI上自习认识一个姑娘，没说过话。下学期有一次在走道里遇到了，就随口问她决定去哪，她说回实习的公司A9。我问A9是什么，她说是Amazon的子公司，偏研究。Wean hall的走廊挤挤嚷嚷，我们只对话了两句，便各自匆匆去上课了。后来，EOC A9来学校Tech talk。那天同时还有Salesforce, 决定去salesforce因为很多学姐学长在那里工作，偌大的教室值熙熙攘攘只坐了几个人，我猜是不发吃的缘故，哈哈哈哈。于是，我也跟朋友也走了，不发吃的留不住人呐（其实不是啦，是学姐跟我说他们公司现在只招实习生）。

于是来到A9现场，屋里坐满了人，我猜食物好吃的缘故。眼看挤不进去， 也快结束了，我就准备走了，结果朋友在门口给我讲这个公司多好多好，在stanford旁边，资源丰富，里面都是PHD。现在想想这助攻的力量真是很大啊，如果你什么的朋友说它好，你也会更容易接受它。于是，这原本不在考虑范围内的公司，我也在网站上投了一下。（Yahoo!的到来让我自动屏蔽了很多没听过的公司，我没有听过的或者内心感觉不如Yahoo的我都没有投过。大概就是曾经沧海难为水吧，它像一个标杆一样，帮我filter掉很多公司）。

大概两周之后，一个周六收到HR的邮件约面试。看地里也没啥面经， 就很快约了电面。

电面1

2道题。Number of islands变种， tree的层级遍历，面得异常好，30min搞定。还走了各种test cases。能听的出来他很满意，说还会有更多的面试，然后我们随便聊聊就结束了。那天面完，心情极好！刚好赶上冰冻了数周的匹村迎来暖阳，然后我穿着最喜欢的裙子从家到PwC开会，一路上高兴感觉要舞起来了。
听上学期面A9的同学说电面一次后就可以onsite了，觉得自己稳了，终于可以再飞加州了，在冷清了3个月之后，所以特别开心。也不是一定要拿下，就是觉得可以出去走走，就很开心。然后，事情并不是想象那样！一周后收到二轮电面，说面js, frontend毕竟这个岗位是data visualization。

电面2

很虚啊，没有很多前端经验，web课上写了一些。但还是兵来将挡的上了，问了react, django。之后又是coding, 给一个regex和一个string, 看是不是valid。比leetcode多了个+操作。 小哥人超nice, 看我一开始stuck住了，特友善的帮我写了很多代码，感觉像他在写，心想完了。主要是我打开leetcode发现自己抄的代码都看不懂思路了（捂脸，我猜我是抄的最佳答案）。 但是，顺着小哥的思路我好怕走不下去。在这个时候，我说我想换个思路，于是就照着抄那个最佳答案，哎，真是艰难，我自己都讲不清思路是什么。最后，我们跑了几个test cases我才绕明白，真是丢脸了。然后，小哥说你没有执行+, 于是我又在代码上修修补补，勉强算糊弄过去了。最后小哥让我问他问题，问了啥我忘了，反正他把我讲得特开心，最后他你这个思路很新，我回去再好好研究研究。

我已经不会有下一步了，确实上一轮表现的踉踉跄跄。但几天后，收到HR邮件说manager要面试我，我心想这是面完直接发offer吗。于是特别take it serious。结果是我想多了，人家是要确认我strong match 才让我上飞机。

电面3

Amazon principle， 工作中的场景问题， test，integration。感觉有几个问题回答的让他很满意，所以这关算是过了。

终于让我飞了！！！

Onsite 6轮轰炸

  1. 美白带个shallow（中国小哥，北京amazon调来的）。JS 所有的false type， false, null, NaN, 0, Undefined。还有啥我忘了，反正当时都猜出来了。然后给一段他们pipeline里的代码，隐去function name让我给他讲讲代码实现了什么，时间复杂度， corner case, 什么情况陷入无限循环。最后，设计videos 的search engine。问了很多feature extraction, 讲每一个feature, 讲的很细，感觉发挥的还不错，想到的全部真诚的表达了出来。
  2. 印度美女。很少见过印度美女，但她是真的长得非常好看那种。还是search engine, 刚其中考试完，对很多知识点还是很熟悉的， NDCG, inverted list, web search vs. product search。最后就写了个找每个word1, word2, …, wordn都出现的docs。都是课上implement过的东西，所以很快写完了。对了，好像有个地方有点问题，她帮我指出来，我夸她很smart，她特别开心的冲我笑。
  3. 中间这个人没印象，博士，搞ML的 。我就很慌了，ML啥都不会，虽然来之前突击了一下，然并卵。还好，我上来先给他打预防针，我没学过，于是我们的探讨就很浅，都是围绕SE 的ML。
  4. manager 带我去吃饭，印度人，就跟他吃玛萨拉了。吃饭最累其实，不停的讲话，回答他的问题，这轮也是judge的。你为什么读MIS不是CS；为什么选择A9；什么让你最骄傲；有没有团队合作中发生冲突，怎么解决；现在都在上什么课；有兴趣学ML吗。。。。。吃完饭，感觉脑子已经不转了。
  5. 第一次面试我的美国小哥，我终于见到真人了。他说我不考java coding了这次。我们一起来来debug吧， js的，内心慌乱。但是他人非常非常好呀，
  6. 光头美国叔叔, 属于企业中层。所以也没问很tech的问题。上来说，面试到现在，你觉得那个问题最有趣，你想让我问你什么问题，挠头。。。。我真是不知道说啥。。。。磨蹭了好一会，他说我还是正经问个问题吧。于是， 他问了我rank diversity。这个topic上课还没讲到，但是凭感觉也能回答，所以也应付过去了。

终于结束了，累死啦。不想张嘴说话。在附近的shopping center 逛了一会就直接去机场了，palo alto 也没什么朋友。坐了夜航，白天直接在学校上课，下午就收到电话说全部positive。

后面，更有趣的故事就是negotiation了，你要想听，我以后慢慢讲。

我在飞机上写这个博客的时候，坐在旁边的西装革履的美国爷爷跟我讲笑话。大部分听不懂，那些文字笑话，但是他还很认真给我解释。我们还没有怎么聊天的时候，他就问我了一个很深刻的问题。你的父母对你最大影响是什么。其实不急思索的，我就回答了他。妈妈很刻苦很执着，我也像她一样继续读书。还有，就是negotiation的技能是从妈妈那里学到的。细细讲讲大概三天三夜都讲不完，孩子是父母的影子。我感觉自己就集合了妈妈的品质爸爸的性格。

美国爷爷是一个投资人。所以我就顺势问了他大公司Amazon为什么要创建子公司A9, AWS。他说当公司的规模扩大，创建子公司可以提神business model的灵活性，子公司更灵活的自我管理，不需要wait大公司approve所有的事情。我觉得他说的很有道理。也没有太多关于这个子公司的疑惑了。

Google Application Engineer面经

坐在从加州到匹兹堡的飞机上，结束了精彩的找工作季。心情还是一如既往的平静和开心。最后一场面试是在Google Sunnyvale office。其实来之前的清晨签下了A9, 所以整个面试旅程没有紧张。但毕竟是曾经的梦想公司，所以还是在酒店简单过了一下系统设计问题。这个岗是Application Engineer, 只能在CA面试。最后，我会详细介绍一下我对这个岗位的了解。

第二天清早打uber, 习惯性的选了拼车，结果正赶上busy hour, 司机要一下子接3个人。结果第一个乘客因为要赶时间，于是跟司机抱怨不能等。司机也是暴脾气，一下子就燃起来了。于是，我就在中间安抚大家。后来总算calm down, 送走了第一个乘客。司机问我是中国人吗，开始跟我讲中文。我很惊讶一个年纪五十多的人英文可以讲的这么溜。他说他是台湾人，但感觉更像北京大老爷们，吐槽湾区的各个公司，Trump的笑话，还讲Google面试不过是2轮脑筋急转弯，吃个饭，再干两轮。很快，我就到了office。

第一轮是美国胖叔叔， 两道题。

  1. 给一个整数数组，是先增后降的。找到peak的值。
  2. 给一个整数set, 找到其中最长的递增子序列，递增是每次加1的那种。

第二轮感觉是中国叔叔。

  1. 给一个整数数组，全部unique. 找到所有和为k的pair.
  2. 设计题目：一个shopping网站， backend的sever只能接受1request/s. 但是frontend会有peak hour 和 steady hour （我暂且这么描述，就是像深夜购物请求次数很少的时间段）。让你设计一个middle layer投balance the requests, smoothly connects frontend and backend.

中午吃饭，日本叔叔，磨磨蹭蹭我们扯东扯西，结果下午面试还晚了一丢丢。

第三轮美国叔叔，纯system design轮。

  1. 他首先问了有没有跟client接触的场合，demo什么的。我想了想，就给他特别详细的将了最近做的Blockchain capstone跟PwC的客户每周开会，讨论进展。他非常认真的记录了三页纸。那会儿，坐在会议室，就感觉特别像在公安局陈述详细的作案过程一样，哈哈哈哈（肯定是最近看嫌疑人X的现身中毒了）。对了，感觉Google的面试官都是经过专业训练过的，中间不会给你很多feedback，一直都看起来很nice的样子，但是不会让你会洞察到他们的心理什么的。每一场面试都是标准题目，然后他们认真记录下你全部是思维过程。剩下的都交给hire committee的人去evaluate。这跟A9和Adobe的面试很不一样，你能感受到他们对你很满意，跟你击掌，不停的夸你什么的。
  2. 设计一个restaurants reservation 网站。跟他讨论了很多方面的问题，他都详细记录下了，然后我们focus在database tables design。 如何记录预定时段，客户，table, 时间， location。

第四轮印度哥哥。

  1. 给一个字符串只包含ABX三种字符，要求把X替换成A/B。要求返回所有可能的组合。
  2. 给一个字符串，要求输出字母按freq降序输出。
  3. ngram的题，很简单的，我忘记具体啥了。

整体来说，没什么难度。Evaluation的事就看hire committee了。决定去A9了，所以也没什么期待。只是打卡式的完成对这个曾经的梦想公司的追逐。其实也怪自己，去年9月HR就联系我面试，我说11月吧， 然后11月的时候我又拖到1月，后来HR没有主动联系我，我就想EOC多面面其他公司，攒足经验再去面Google, 结果春季的招聘机会烧的让人绝望。2月Google说SE已满，本来约的onsite被取消了，后来变成了AE。

跟AE的工程师和好朋友聊了一下，说说我对这个岗位的理解。这个岗位是做internal tools的，client是公司内部人，比如HR, ERP, Financial 系统的整合与提升。30%的时间可能会跟各个部门的人communication, 写design docs，项目很灵活，可以接触到很多projects。原先，这个岗位有很多part-time, contractor来做，现在公司规模扩大，想自己搞起来。今年，很多朋友都走了AE的岗。工资方面比SE低一点，大概11W, signon可以argue到一个不错的level 看个人negotiation的能力了。

总体来讲，Google还是有很多很好的资源，食堂的饭各种各样， café, 零食，坚果到处都是。可以上很多的课程，一起学习，健身房，医疗什么的。只是campus太大，即使同在一个楼，可能大部分人都不太认识，然后很多开会还是靠视频。我只是走马观花很浅显的了解，毕竟还没工作。

Yahoo面经

面经这东西应该当时记下来，可惜太懒了一直拖到现在。记忆也不是很清晰，只能尽力回忆，希望对大家有帮助。

Yahoo Onsite之前有两轮电面：

电面1：

  1. The number of islands
  2. palindrome

电面2：

这个就很有意思了，问了一些DS的问题, OSI, network, web protocol (TCP/UDP). 然后准备写code。我说还在codeedit上写吗？第一次电面创建了一个文档，还在哪里吗？然后他也进入那个文档，发现我上次已经写了两个题。然后，他就不让我写code了。于是，我们聊了一下load balancer，partition/sharding CC的东西。他又问我了一些linux的问题。后来，我讲到最近在做513 singal shell的作业。这之后，他跟我聊了半小时我是怎么实现的，非常细，细到每一行code, 幸好我很熟啊， 昨天晚上搞到12点才弄完，记忆非常清晰，就跟他讲了很多很多。然后就结束了。

电面很久之后也没什么消息。而我上学期5门口，能力不足忙到焦头烂额。每周都小心的规划着先赶什么due赶到什么程度再搞下一个，周末也很愉快的在补作业中度过了。大概过了3、4周的样子，那天上午去学校的路上我还跟学姐聊下学期找工作来不来得及，Yahoo！没理我种种。然后到学校就收到了onsite邀请，正赶上期末前，舍弃2个in-class quiz，我还是犹豫要不要去。后来，想想还没去过硅谷，也有点想离开目前生活的小bubble去外面看看， 就答应了。HR很不给力，订机票酒店他不主动，我也没管，知道出发前一天，我想她再不给我订，我就不去了， 我也没有任何准备其实。给HR发了邮件，她很快安排好一切，就这样，出发前一天，才安顿下心情去面试。然后我在一亩三分地上搜了一下面经，DNS组好像只有1篇面经，总共Yahoo!也没多少面经，我把最近2年的看了一下，就到硅谷了。

房间有两个双人大床，开心的我蹦来跳去不知道睡哪个。因为没有一定要拿下的心态，也没有调时差，晚上8点就睡了，早晨4点醒了。发现Terry lab在面试完刚好due了，于是早上还在搞lab, 还好极其简单。然后有扫了一遍题目，去Yahoo!了。

面试流程非常有条理，先在前台汇集，看一堆来面试的人， 大家面目紧张，也不说话。我就先说话了，于是也认识了几个好朋友，然后HR warm up全场，大概20多人(因为临近年末，招聘尾声，就没有以前那么多)。发现3个CMU的也是开心，然后其他来着伯克利，斯坦福，哥大，密歇根等等。虽然感觉每天在CMU虐的自信全无，出来溜溜觉得自己还挺骄傲自信的。然后HR带着我们浏览Yahoo! Campus， gym, café，真的很不错，比较老牌大公司，还是设施很全很多的。最后我们被带到一个building 吃早餐，等待面试官把我们领走。

本来安排4轮，后来变成3轮了。

第一轮

好像sliding window, 取最大值； 好像用到hashmap，具体题目，我记不起来了，I am sorry! 记得hashmap是因为Terry 课上刚implement hashmap, 然后我当时脑子应该是锈住了，一个劲儿的讲collision啥的，还好面试官把我带回来了（哎，当时胡说了那么多， 人家还是让我过了，真是心善）。最后我们讨论的很愉快，他说我的方法很新颖，这问题是他之前工作中遇到的。

第二轮美国哥大小哥。

那天他在家办公。于是我们就愉快的video interview了，好像也是explore islands的变种。很简单，我写得很快，有一个小错了，他提醒了我，然后就愉快的唠嗑了。

Stanford statics/ML PHD, 国人。

这一轮我真是面得一塌糊涂。先问问Exception, 刚好答反。然后， 他准备问我ML，我吓坏了，连忙解释说自己没学过。大概他也只准备了一个问题吧，看他很为难，想了很久。后来他问我一个简单的场景问题，如何预测trending words. 听力不好，以为是Trading, 说了一大堆金融交易的废话，看他表情不太对，我马上刹车。修正方向，继续胡侃，中间他问了很多问题，具体实施，我是一点经验也没有，就尽力去想，一直讨论，一直修正。最后，他说结束了，我想我完了。下电梯他开始跟我讲中文，他问我在北京上学，我点点头。然后我问他为什么选择在Yahoo!, 他说做ML, Yahoo!提供很多大数据，而且技术也是业界通用的， 不像Google好多内部开发，不对外。 很快，他带我回到了最初的大楼。我是最早回去的interviewee, 等大家陆续回去， 每个座子（4个interviewee）也坐个manager。一开始气氛比较冷清，后来大家瞎聊。坐我旁边的中国小哥跟我说中文，现在想想这样真不好，我们把经理晾在一边。不过，我还是尽力跟manager聊很多，关于加州的生活，房租什么的。

然后就飞回来继续在CMU受苦，大概两周之后收到offer。当时也没有很激动，觉得自己还没有开启找工作模式，刷题也不到100道，不能就止步于此 。当时每次电面都无准备的直接上，其实约电面的时候我也推三阻四很难找到空闲的时间。当时想法很简单，就想着寒假好好刷题，下学期轰轰烈烈的找工作。 感觉自己还是很幸运的，就凭着coding bootcamp, 513和DS学的三脚猫知识，无心插柳般的拿到offer, 没任何紧张，焦灼的等待过程。后来， 下学期找工作的寒冬时代无数次的证明证明这个offer对于稳定我心有多么多么重要，我很感激Yahoo!.
